Usage:

Download files 

  * Either with github website or git clone CLI

Have python

Have Django

  * CLI: pip install django

Start server:

  * cd to folder containing manage.py then

  * CLI: python manage.py runserver 8000
  
  * note: 8000 is a semi-arbitrary port

Check it out:

  * open browser
  
  * browse to localhost:8000/
